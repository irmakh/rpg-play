# Context Directory

> Reference material and domain knowledge

## Purpose

Context files contain static information the system uses to reason and make decisions.

## Examples

- Tone/voice guidelines
- Writing samples
- ICP (Ideal Customer Profile) descriptions
- Case studies
- Design principles
- Negative examples (what NOT to do)
- Domain-specific knowledge

## Format

Use markdown files organized by topic:

```
context/
├── voice_guidelines.md
├── examples/
│   ├── good_example.md
│   └── bad_example.md
└── domain/
    └── terminology.md
```

## Usage

Context shapes quality and style of outputs. The AI references these files when making decisions about how to approach work.
